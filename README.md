### Тестовое задание для LeadHit

#### **Как пользоваться**:

*Версия Python: 3.11.6*

Для скачивания необходимых пакетов
```
pip install -r requirements.txt
```
Для запуска веб-сервера
```
uvicorn main:app --reload
```
Файл для тестовых запросов
```
python req.py
```
